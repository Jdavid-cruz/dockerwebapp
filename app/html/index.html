MI PROYECTO DOCKER

SOBRE MÍ

Soy un administrador de sistemas con experiencia en Linux y en la gestión de aplicaciones en contenedores Docker y Nginx. Me apasiona optimizar sistemas y gestionar infraestructuras eficientes.

SOBRE EL PROYECTO

Este proyecto consiste en crear un entorno de desarrollo usando Docker y Nginx para desplegar una aplicación web sencilla. A continuación, se describen los pasos realizados:

  1. Creación de un archivo Dockerfile y docker-compose.yml.
  2. Configuración de un servidor web con Nginx.
  3. Implementación de un archivo HTML y CSS para la presentación de la página.
  4. Integración de la aplicación con Docker para facilitar el despliegue y la gestión de contenedores.
